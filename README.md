# RofiTodo

A to-do list using Rofi

## Installation

1) Clone this repository
2) Go to `Rofitodo`
3) Run the following commands

```bash
autoreconf -si
./configure
make
make install
```

You can also build RofiTodo using cargo with

```bash
cargo build --release
```

and put the `rofitodo` executable (located in `target/release`) where you want.
